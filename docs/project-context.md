# Olympic Data Story - Contexto del Proyecto

## Objetivo
Crear una web narrativa de visualizacion de datos olimpicos con estilo editorial inspirado en Obys, usando un menu principal con 4 historias y navegacion a paginas con scroll explicativo.

## Restriccion clave
Todo el stack debe ser gratuito (open source o free tier).

## Stack acordado (gratis)
- Next.js + TypeScript + App Router
- Tailwind CSS
- D3.js / Observable Plot para visualizaciones
- GSAP o Framer Motion para animaciones
- Lenis + scroll narrativo progresivo
- Datos en CSV/JSON preprocesados

## Historias definidas
1. Cold War in Gold - La guerra que no fue guerra
2. El Atlas del Cuerpo Olimpico
3. El Cementerio Olimpico
4. 10 Olimpiadas, Una Vida

## Estructura base creada
- Home con menu y enlaces a las 4 historias
- Ruta por historia con plantilla de contenido
- Carpetas para componentes, datos, contenido y utilidades

## Rutas actuales
